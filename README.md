# ChallengeAIlabAeroespacial

### Server Installation
```
pip install virtualenv
python -m venv venv
venv\Scripts\activate		# For Windows
source ./venv/bin/activate	# For Linux
pip install -r dependencies.txt
```

### Server Run
```
venv\Scripts\activate		# For Windows
source ./venv/bin/activate	# For Linux
flask run
```
